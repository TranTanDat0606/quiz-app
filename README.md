# React Quiz App

- react typescript
- redux
- export csv
- MUI

## Requirement

1. Dashboard page
   - form input
     - validate
     - navigate to Question page when click submit
2. Question page
   - Show Question List
   - Count down
   - Auto switch orther question when countdown === 0
   - Increase scrose when selecting answer
3. Final Score page
   - form input
     - validate
4. Leaderboard page
   - Export CSV
   - Show list user

- get API category: [https://opentdb.com/api_category.php]
- get question API: [https://opentdb.com/api.php?amount=2&category=13&difficulty=medium&type=multiple]
